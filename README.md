# 대지좌표 정수보정 (Artboard Coordinate Fixer)

일러스트레이터에서 PNG로 내보낼 때, 대지 좌표가 소수점이면 원본보다 1px 더 크게 저장되는 버그가 있다. 배너 광고 등록 시스템처럼 사이즈가 정확히 안 맞으면 오류가 나는 곳에 올릴 때 특히 문제가 된다.

이 스크립트는 문서 안 모든 대지의 좌표를 정수로 보정한다.

- 대지가 이동한 만큼 안에 있는 오브젝트도 같이 이동시켜, 대지와 그림이 어긋나지 않게 한다
- 문서에 배율(Scale Factor)이 걸려있어도 자동 감지해서, 화면에 보이는 값 기준으로 정확히 보정한다 (일러스트레이터는 대지 한 변이 227인치를 넘으면 자동으로 배율이 걸리는데, 이걸 모르고 반올림하면 크기가 엉뚱하게 튀어버린다)

## 사용법

1. 대상 파일을 일러스트레이터에서 연다
2. `File > Scripts > Other Script...` 로 `대지좌표_정수보정.jsx` 를 불러온다
3. 완료되면 몇 개 대지를 보정했는지, 감지된 배율이 무엇인지 알림창으로 알려준다

## 설치 (매번 찾기 귀찮다면)

```
Applications/Adobe Illustrator [버전]/Presets.localized/[언어 폴더]/Scripts/
```
위 경로에 복사해두면 `File > Scripts` 메뉴에 항상 뜬다.

## 만든 과정

실제로 겪은 시행착오(1px 버그 → 그림 어긋남 → 975가 980으로 튀어버린 배율 미스터리)는 블로그에 정리해뒀다.
